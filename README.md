# 🍓 Vynica — Your All-in-One Discord Companion

Vynica is a versatile, multi-purpose Discord bot built to replace the 10 different bots you'd normally need. Image enhancement, a full economy system, cross-server chat, moderation, relationships, and more — all in one place.

## ✨ Features

| Category | What it does |
|---|---|
| 💞 **Relationships & Fun** | `+marry`, `+adopt`, SFW roleplay commands, truth or dare |
| 🎲 **Economy** | Work for cash, gamble with coinflip/slots, rob other users, spend in the shop |
| 🔗 **Cross-Server Chat** | Connect and chat with users in other servers or 1-on-1, with emoji/avatar support |
| 🛡️ **Moderation** | Automod that filters offensive language, bad words, and spam links |
| 🎨 **Creativity & Utility** | Cross-server emoji use, image quality enhancement, meme/sticker generation, gif effects |
| 🍁 **Extras** | Welcome/goodbye messages, premade rules, quizzes, mini-games |

**Prefix:** `+` · **Help:** `+anyahelp`

---

## 🛠️ Tech Stack

- **Language:** JavaScript (Node.js)
- **Framework:** [Discord.js](https://discord.js.org/) v14
- **Package Manager:** npm
- **Database:** SQL / SQLite for persistent data (economy, relationships, moderation logs)
- **Data Storage:** JSON for lightweight config and static data
- **Environment Config:** dotenv (`.env`)
- **Image Processing:** Canvas / Sharp / Gif-Encoder-2 for image enhancement, filters, and gif effects
- **Hosting:** _(e.g. Railway / VPS / Pterodactyl — adjust to your setup)_
- **Website:** PHP, HTML, CSS, JavaScript

## 🚀 Using Vynica

Vynica is a hosted bot — no setup required. Just invite it to your server and start using commands with the `+` prefix.

1. [Invite Vynica to your server](https://discord.com/oauth2/authorize?client_id=1182053086513074207)
2. Run `+anyahelp` to see all commands
3. Check out the full **[command list](https://vynicabot.site.je/commands/)** and **[website](https://vynicabot.site.je/)** for setup guides, feature walkthroughs, and screenshots
